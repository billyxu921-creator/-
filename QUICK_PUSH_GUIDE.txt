⚠️  第一步：安装Xcode命令行工具（必须！）

  xcode-select --install
  
  等待5-10分钟完成安装

🚀 第二步：推送到GitHub
───────────────────────────────────────────────────────────────
  方式A：自动化脚本（推荐）
  
    ./push_to_github.sh
  
  方式B：手动执行
  
    git init
    git add .
    git commit -m "Initial commit: A股投资分析自动化系统"
    git remote add origin https://github.com/billyxu921/股票助手.git
    git branch -M main
    git push -u origin main
