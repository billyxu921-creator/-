# 数据源说明文档

## 系统数据来源详解

本系统所有数据均通过 **akshare** 库从公开网站获取，不涉及任何内幕信息或非公开数据。

---

## 1. 股吧/新闻数据

### 数据源1: 东方财富股票新闻

**接口**: `ak.stock_news_em()`

**数据来源网站**: 
- 东方财富网: http://finance.eastmoney.com/
- 新闻频道: http://finance.eastmoney.com/news/

**返回字段**:
- 关键词 (股票代码)
- 新闻标题
- 新闻内容
- 发布时间
- 文章来源
- 新闻链接

**示例URL**:
```
http://finance.eastmoney.com/a/202601083612903...
http://finance.eastmoney.com/a/202601083612970...
```

**数据特点**:
- 实时更新
- 包含市场热点
- 涵盖个股新闻
- 公开可访问

---

### 数据源2: 主力资金流向新闻

**接口**: `ak.stock_news_main_cx()`

**数据来源网站**:
- 东方财富资金流向: http://data.eastmoney.com/zjlx/

**返回字段**:

## akshare 库说明

### 什么是 akshare？

akshare 是一个开源的金融数据接口库：
- GitHub: https://github.com/akfamily/akshare
- 文档: https://akshare.akfamily.xyz/
- 许可证: MIT License

### 工作原理

```
公开网站 → akshare爬虫 → 结构化数据 → 我们的系统
```

akshare 负责：
1. 从公开网站抓取数据
2. 清洗和格式化数据
3. 提供统一的API接口

我们的系统负责：
1. 调用akshare接口
2. 分析和处理数据
3. 生成投资报告

## 数据使用限制

### 请求频率

为避免对数据源造成压力：
- 建议间隔 ≥1秒
- 批量请求使用延时
- 避免高频重复请求

### 数据缓存

建议实施缓存策略：
- 基本信息: 缓存1天
- 实时行情: 缓存1分钟
- 股东信息: 缓存1周
- 历史数据: 永久缓存

### 合规使用

- ✅ 个人研究使用
- ✅ 投资决策参考
- ❌ 商业转售数据
- ❌ 高频交易套利
- ❌ 市场操纵行为

## 数据源可靠性

### 优点

1. **官方数据**: 来自交易所和监管机构
2. **实时更新**: 数据及时准确
3. **免费使用**: 无需付费订阅
4. **接口稳定**: akshare维护良好

### 局限性

1. **网络依赖**: 需要稳定网络连接
2. **接口变更**: 数据源可能调整接口
3. **访问限制**: 可能有反爬虫机制
4. **数据延迟**: 非专业行情源

## 技术支持

### 数据问题排查

1. **无法获取数据**
   - 检查网络连接
   - 更新akshare版本
   - 查看错误日志

2. **数据不准确**
   - 对比多个数据源
   - 检查时间戳
   - 确认股票代码

3. **接口报错**
   - 查看akshare文档
   - 检查参数格式
   - 降低请求频率
