# 快速开始指南

> 5分钟上手全网热点发现引擎

## 🚀 三步启动

### 第1步：安装依赖

```bash
# 安装Python包
pip3 install playwright pandas requests

# 安装浏览器驱动
playwright install chromium
```

### 第2步：检查配置

确保 `config.py` 文件存在且包含DeepSeek API Key：

```python
DEEPSEEK_CONFIG = {
    'api_key': 'sk-8b60ff11aefd4032a572f736087f175f',
    ...
}
```

### 第3步：运行程序

**方式1：使用启动脚本（推荐）**
```bash
./run_discovery.sh
```

**方式2：直接运行Python**
```bash
python3 Discovery_Engine.py
```

**方式3：运行测试（减少抓取量）**
```bash
python3 test_discovery_engine.py
```

## ⚠️ 常见问题

### Q1: 浏览器无法启动

**解决方案**:
```bash
playwright install chromium
```

### Q2: 需要登录怎么办？

**操作步骤**:
1. 在浏览器窗口中扫码或输入账号密码
2. 等待登录成功
3. 在终端按 **回车** 继续

### Q3: 遇到验证码怎么办？

**操作步骤**:
1. 程序会发出提示音（🔔）
2. 在浏览器中手动完成验证码
3. 完成后在终端按 **回车** 继续

### Q4: 抓取数据为空

**可能原因**:
- 网络问题
- 页面结构变化
- 被反爬拦截

**解决方案**:
- 检查网络连接
- 增加等待时间
- 重新运行程序

### Q5: AI分析失败

**解决方案**:
```bash
# 测试API连接
python3 test_api_key.py
```

## 🎯 使用建议

### 1. 最佳运行时间
- **推荐**: 每天早上9点
- **原因**: 数据更新完整，市场开盘前

### 2. 运行频率
- **推荐**: 每天1次
- **原因**: 避免被反爬拦截，积累历史数据

### 3. 数据积累
- **首次运行**: 没有昨日数据，对比效果有限
- **连续3天**: 开始有对比效果
- **连续7天**: 数据充足，分析准确

### 4. 结果解读
- **增长率 > 200%**: 高度关注
- **置信度 > 0.7**: 可信度高
- **结合多个数据源**: 交叉验证

## 📚 进阶使用

### 1. 自定义抓取数量

编辑 `Discovery_Engine.py`：

```python
# 修改main()函数中的参数
engine.run(
    xhs_count=100,  # 改为100条
    headless=False
)
```

### 2. 定时自动运行

使用cron定时任务（macOS/Linux）：

```bash
# 编辑crontab
crontab -e

# 添加定时任务（每天早上9点）
0 9 * * * cd /path/to/project && python3 Discovery_Engine.py
```

### 3. 查看历史数据

```bash
# 查看历史数据目录
ls -lh discovery_history/

# 查看某天的数据
cat discovery_history/discovery_20260115.json
```
