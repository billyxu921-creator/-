# 量化选股器 (Quant Picker)

> AI驱动的量化选股系统 - 技术面 + 舆情面 + AI决策

---

## 🎯 核心功能

**Quant Picker** 结合量化指标、社交媒体舆情和AI分析，自动筛选出最具潜力的股票。

### 4步选股流程

```
Step 1: AkShare指标初选
  ↓ 筛选条件: 60<市值<200亿, 2%<涨幅<6%, 换手率>5%
  
Step 2: 舆情碰撞
  ↓ 匹配微博+小红书热点，百万粉丝博主×10权重
  
Step 3: DeepSeek终极筛选
  ↓ AI从TOP 10中选出TOP 3，预测止盈位
  
Step 4: 生成AI推荐报告
  ↓ 输出Markdown格式的详细分析报告
```

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install akshare pandas numpy requests
```

### 2. 配置API Key

确保 `config.py` 中已配置DeepSeek API Key。

### 3. 准备舆情数据（可选）

```bash
# 抓取微博舆情
python3 weibo_sentiment_weighted.py

# 抓取全网热点
python3 Discovery_Engine.py
```

### 4. 运行选股器

```bash
python3 Quant_Picker.py
```

## 🎯 筛选标准

### Step 1: 量化指标

| 指标 | 范围 | 说明 |
|------|------|------|
| 市值 | 60-200亿 | 中小市值，成长空间大 |
| 涨幅 | 2%-6% | 有一定涨幅，但未过热 |
| 换手率 | >5% | 活跃度高，资金关注 |

### Step 2: 舆情评分

```
舆情评分 = (基础分50 + 关键词加分) × 影响力权重

关键词加分:
- 高优先级 (重组/利好/涨停): +30分
- 中优先级 (增持/回购/业绩): +20分
- 低优先级 (关注/看好): +10分

影响力权重:
- 100万+粉丝: ×10
- 10万+粉丝: ×3
- 其他: ×1
```

### Step 3: 综合得分

```
综合得分 = 涨幅×30% + 换手率×20% + 舆情评分×50%
```

## 🐛 常见问题

### Q: 初选股票数为0？
A: 可能是筛选条件过严或非交易时间，建议放宽条件或在交易时间运行。

### Q: 未找到舆情数据？
A: 先运行 `weibo_sentiment_weighted.py` 和 `Discovery_Engine.py` 抓取舆情数据。

### Q: AI分析失败？
A: 检查DeepSeek API Key配置和网络连接。
